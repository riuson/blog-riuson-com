BB & Qt — Установка Qt Creator

15 Июль 2014 г., Вт

Подготовка Qt Creator для сборки проектов под BeagleBone.

Установка

Для установки Qt Creator берём Qt Online Installer. С его помощью загружаем Tools и Qt 5.2.1 gcc 32/64 bit.

Далее запускаем свежеустановленный Qt Creator.

Добавление устройства

Открываем меню Иструменты -> Параметры -> Устройства. Добавляем новое "Обычное Linux-устройство", называем его "BeagleBone Black - Debian". Хост 192.168.7.2, авторизация по паролю, логин debian, пароль temppwd.

Добавление комплекта

Открываем меню Иструменты -> Параметры -> Сборка и запуск.

На вкладке Отладчики добавляем новый отладчик Linaro GDB ARM Python, указывая путь к ранее собранному Linaro GDB: /home/user/beaglebone/gdb-linaro-7.6-2013.05/bin/arm-elf-linux-gnueabihf-gdb

На вкладке Компиляторы добавляем новый кросс-компилятор Linaro GCC ARMv7, указывая путь к копилятору: /home/user/beaglebone/gcc-linaro-arm-linux-gnueabi-4.8-2014.03_linux/arm-linux-gnueabihf/bin/g++

На вкладке Профили Qt добавляем новый профиль Qt 5.2.1 (armv7), указывая путь к qmake, из собранного ранее Qt 5.2.1 для ARMv7: /usr/share/qt-5.2.1-armv7/bin/qmake
